cmCsvUpload
directive in module _directives

Description

Read csv file

Dependencies

  • Lodash
  • $log
  • $rootScope
  • $timeout
  • ngDialog
  • cmNotify
  • gettextCatalog

Usage

as element:
<cm-csv-upload
       csv-result="{Object}"
       csv-header="{Object}"
       csv-name="{Object}"
       callback="{Function}">
</cm-csv-upload>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
csvResultObject

Passing object csv result to parent

csvHeaderObject

Passing object csv header to parent

csvNameObject

Passing csv filename to parent

callbackFunction

Callback function (optionnal)

Events

  • change

    On change, parse the csv file with Papaparse
