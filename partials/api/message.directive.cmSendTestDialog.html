cmSendTestDialog
directive in module message

Description

Directive to create a message

Dependencies

  • $log
  • apiMessage
  • cmNotify
  • gettextCatalog

Usage

as element:
<cm-send-test-dialog
       url-message="{String}"
       on-success="{Function}">
</cm-send-test-dialog>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
urlMessageString

Message url to call api and launch email test

onSuccessFunction

success callback function, after the dialog success

Methods

  • sendTest()

    Send the test message to recipients

Properties
