cmFormRegen
directive in module applications

Description

Update application or regen secret

Dependencies

  • apiApplications
  • cmNotify
  • Lodash
  • $log
  • gettextCatalog

Usage

as element:
<cm-form-regen
       app="{Object}"
       regen-url="{Object}"
       on-cancel="{Function}">
</cm-form-regen>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
appObject

Passing object application smtp or api

regenUrlObject

Passing url to regen secret

onCancelFunction

Cancel function to close dialog window
