cmApplications
directive in module applications

Description

Directive to display applications

Dependencies

  • apiApplications
  • cmNotify
  • Lodash
  • $log
  • gettextCatalog
  • ngDialog

Usage

as element:
<cm-applications>
</cm-applications>

Methods

  • delete()

    Delete an application

  • showDialogApi()

    Open a ngDialog to create a new api application

  • showDialogSmtp()

    Open a ngDialog to create a new smtp application

Properties

  • getApiApp

    Init a array which contain all api applications

  • getSmtpApp

    Init a array which contain all smtp applications
