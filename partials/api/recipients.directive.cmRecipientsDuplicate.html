cmRecipientsDuplicate
directive in module recipients

Description

Display to user a dialog with all duplicate recipients

Dependencies

  • $log
  • ngDialog
  • Lodash

Usage

as attribute
<ANY cm-recipients-duplicate
     duplicate="{Object}"
     callback="{Function}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
duplicateObject

Passing the array with duplicate recipients

callbackFunction

the callback function with recipients selected

Methods

  • changeRadio()

    Set all inputs radio value to false

  • close()

    Close all dialog

  • passingSelected()

    Get selected recipients and passing them into the callback

  • showNext()

    Get the next 20 recipients in duplicate
