Crunchmail Zimlet iFrame

Il s'agit du code qui est chargé dans une iFrame dans un onglet du Webmail Zimbra. Pour le zimlet lui même, c'est ici : https://git.owk.cc/crunchmail/zimlet

Requirements

  • Install Node
    • on OSX install home brew and type brew install node
    • on Windows install chocolatey and type choco install nodejs
  • On OSX you can alleviate the need to run as sudo by following these instructions. I highly recommend this step on OSX
  • Open terminal
  • Type npm install -g gulp

Installing npm dependencies

  • Open terminal
  • Type npm install

Installing bower dependencies

  • Open terminal
  • Type bower install

Running

gulp

Il est aussi possible d'ajouter --prod pour forcer certaines étapes spécifiques à la prod, comme le minify JS. Ensuite aller sur http://localhost:4001

Structure

All gulp tasks are specify on gulp-tasks folder. Paths declared in gulp tasks are in gulp.config.json file.

This application is separated in several modules. Every modules is an application feature. Go to https://github.com/crunchmail/frontend-assets to see the structure of assets and https://github.com/crunchmail/frontend-directives for the structure of directives

/src
    /app
    /assets
    /components
        /frontend-directives
        /frontend-assets
    /iframe_integration
    /static
